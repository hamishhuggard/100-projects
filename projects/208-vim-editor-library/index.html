Vim Editor Help

Features

  • Full Vim Keybindings: All standard vim commands work
  • Syntax Highlighting: Support for multiple programming languages
  • Multiple Themes: Choose from various editor themes
  • File Operations: Save, load, and create new files
  • Real-time Status: Shows current mode, cursor position, and file info

Basic Vim Commands

  • i, a: Enter insert mode
  • Esc: Return to normal mode
  • h, j, k, l: Move left, down, up, right
  • w, b: Move by word
  • 0, $: Beginning/end of line
  • gg, G: First/last line
  • dd: Delete line
  • yy: Copy line
  • p, P: Paste
  • u: Undo
  • Ctrl+r: Redo
  • v: Visual mode
  • Shift+V: Visual line mode
  • /: Search
  • n, N: Next/previous search result

Controls

  • Language: Select syntax highlighting mode
  • Theme: Choose editor color scheme
  • Save: Download current content as file
  • Load: Load file from your computer
  • New: Clear editor and start fresh
