Vim Editor Help

Modes

  • Normal Mode: Default mode for navigation and commands
  • Insert Mode: Type text (press 'i', 'a', 'o', 'O')
  • Visual Mode: Select text (press 'v')
  • Exit Insert/Visual: Press 'Esc' or 'Ctrl+C' to return to normal mode

Basic Navigation

  • h, j, k, l: Left, Down, Up, Right
  • w, b: Next/Previous word
  • 0: Beginning of line
  • ^: First non-blank character of line
  • $: End of line
  • gg, G: First/Last line

Editing

  • i, a: Insert before/after cursor
  • o, O: New line below/above
  • dd: Delete line
  • yy: Copy line
  • p, P: Paste after/before
  • u: Undo
  • Ctrl+r: Redo

Visual Mode

  • v: Enter visual mode (character selection)
  • Shift+V: Enter visual line mode (line selection)
  • y: Copy selected text
  • d: Delete selected text
  • c: Delete selected text and enter insert mode
  • Esc/Ctrl+C: Exit visual mode

Commands

  • :w - Save file
  • :q - Quit
  • :wq - Save and quit
  • :new - New file
  • :set filename.ext - Set file type for syntax highlighting
  • :help - Show this help

Search

  • / - Search forward
  • n, N: Next/Previous search result
