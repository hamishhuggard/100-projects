# Elliptical Orbit

An interactive visualization of Earth's elliptical orbit around the Sun with adjustable eccentricity.

## How to Run

You can run this project in two ways:

1. **Simple method**: Open `index.html` directly in your web browser
2. **Local server method**: Run `python -m http.server` in this directory, then open `http://localhost:8000` in your browser

## About

This project demonstrates how Earth orbits the Sun in an elliptical path. Use the eccentricity slider to see how the orbit shape changes from nearly circular to highly elliptical.
